Process window

Process windows can be created from the Debug view. They persist after termination of a simulation and are reused when restarting a simulation. Process windows are only visible in the POOSL Debug perspective.

A process window consists of 2 views:

  1. Execution Tree view
  2. Variables view
In a process window, these views are restricted to a specific process. The title of the window consists of the model name and the process name. Multiple process windows can be used to monitor multiple processes at the same time.
