Create POOSL Projects and Models

Create a POOSL Project

Follow these steps:

  1. Click on File -> New -> Project... and select "POOSL Project".
  2. Click Next, and type a name for the project.
  3. Click Finish to create the POOSL Project.
The created POOSL Project has one folder "models", which contains a POOSL Model. Eclipse switches automatically to the POOSL Edit perspective.

Create a POOSL Model

After creating a POOSL Project, follow these steps:

  1. Right click the folder "models" in the POOSL project, and select New -> POOSL Model.
  2. Click Next, and type a name for the model.
  3. Click Finish to create the POOSL model.
After creating a POOSL model, continue with editing.
