Importing and Exporting legacy XML

Installing legacy support

The support for importing and exporting legacy Rotalumis and SHESim xml files is part of a separate plugin. This plugin can be installed from the Poosl update-site: "http://poosl.esi.nl/downloads/ide/updates/release" by selecting "Poosl" -> "Legacy Support for POOSL IDE".

Importing a legacy SHESim XML file

To create a POOSL file from a legacy SHESim XML file, follow these steps:

  1. Click on File -> Import...
  2. In the Import window select POOSL -> "Import Legacy XML as Poosl" and click Next .
  3. In the "File import Wizard" select the legacy XML file by browsing to it.
  4. Select the destination location, give the new file a name and click Next.

Exporting a legacy SHESim XML file

The generation of a legacy Rotalumis XML file from a POOSL file can be initiated in 2 ways:

  1. Right click on the POOSL file in the Project Explorer;
  2. Right click in the text editor when the POOSL file is opened.
Then select "Generate Legacy SHESim XML" from the context menu. The generated files will be put in the "simulator" folder at the same location as the original file.
