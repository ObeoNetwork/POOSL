Preferences

General Eclipse preferences

Settings like font properties can be found as follows:

  1. Go to Window -> Preferences
  2. Select General -> Appearance -> Colors and Fonts
Settings like line numbers and spaces/tabs can be found as follows:
  1. Go to Window -> Preferences
  2. Select General -> Editors -> Text Editors

POOSL specific preferences

POOSL specific settings can be found as follows:

  1. Go to Window -> Preferences
  2. Select POOSL
The options to set preferences for are:
  • Basic Classes: Expert option for changing the basic classes.
  • Editor: Preferences about automatically switching to the POOSL Edit perspective, auto-completion of brackets/quotes and auto-format on save.
  • Graphical Behavior: Settings for the behavior when opening Poosl models from the project explorer and graphical editor.
  • Refactoring: General preferences about refactoring.
  • Sequence Diagram: Settings for the Sequence Diagram view.
  • Simulator: Determine the logging level of the communication with the simulator. This is only used for bug reporting. See http://poosl.esi.nl for more details.
  • Syntax Coloring: Change the font, size and color of comment sections, keywords etc.
