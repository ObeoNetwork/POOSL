Exchange and Backup POOSL Models and Diagrams

The graphical POOSL diagrams are not stored inside a poosl file, but in the representations.aird file. Each project has one representations.aird file that contains all diagrams from the project.

Exchanging POOSL models including their diagrams

The easiest way to exchange models and diagrams with other POOSL users is as follows:

  1. Create a new POOSL project in the same workspace as the original project
  2. Copy (including diagrams) the file(s) or folder(s) from the original project
  3. Paste (including diagrams) the file(s) or folder(s) into the new project
  4. Export the project from the current workspace
  5. Import the project into someone the other POOSL users' workspace
