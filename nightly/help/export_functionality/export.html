Importing and exporting archives

Importing a project from an archive

To create a POOSL project from an archive (zip/tar) file, follow these steps:

  1. Click on File -> Import... -> General -> Existing Projects into Workspace.
  2. Click on Select archive file, and press Browse... to select the archive file.
  3. Click Finish to import the project.

If the imported project contains a file "representations.aird" with graphical layout information, we recommend that you manually check the following setting. In the top-right corner of the Project Explorer, click on the downward-pointing little arrow ("View Menu"), and select "Customize View...". In tab "Content", deselect the extension "Sirius Content Management"

Exporting a project as an archive

To export a whole POOSL project as an archive (zip/tar) file, perform these steps:

  1. Right-click on the project in the Project Explorer
  2. Select Export... -> General -> Archive File
  3. Click Browse to select the destination file
  4. Click Finish
